# Eseosa's Smart Utility Toolkit

A multi-functional mobile application built with Flutter that combines essential daily tools into one seamless experience. This project was developed as part of the HNG Internship Stage 1 task.

## 🚀 Features
- **Unit Converter:** Easily convert between different units (Length, Weight, etc.).
- **Task Manager (To-Do):** - Create, Edit, and Delete tasks.
  - Mark tasks as completed with a strike-through effect.
  - **Local Persistence:** All tasks are saved locally using `shared_preferences`, so your data remains even after closing the app.
- **Offline Mode:** The app works fully without an internet connection.

## 🛠️ Tech Stack
- **Framework:** [Flutter](https://flutter.dev/)
- **Language:** [Dart](https://dart.dev/)
- **Data Persistence:** [shared_preferences](https://pub.dev/packages/shared_preferences)
